## Инструкции по запуску

```
$ sudo apt-get install mongodb
$ git clone https://github.com/MaxSamokhin/chat.git
$ cd chat
$ npm install
$ npm run start-back
$ npm run start-front
$ npm run build
```
Приложение станет доступно на [localhost:8001](http://localhost:8001)
